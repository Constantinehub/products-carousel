# Products Carousel APP - React + TypeScript + Vite

## Instruction for running the app locally

1. **Clone the repository**

   ```
   git clone https://github.com/Constantinehub/products-carousel.git && cd products-carousel
   ```

2. **Install dependencies**

   ```
   npm install
   ```

3. **Start development server**

   ```
   npm run dev
   ```

   The app will be available at [`http://localhost:5173`](http://localhost:5173)

4. **Build for production (optional)**
   ```
   npm run build
   ```

5. **Or you can enjoy the [`DEMO`](https://constantinehub.github.io/products-carousel/)**
